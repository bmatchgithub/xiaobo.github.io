用hexo搭建自己的博客(无脑版)

Posted on 2020-03-15 | In hexo

关于hexo的一些使用体会

前两天学习了一手hexo搭建自己的blog网站，感觉还挺有意思的，接下来我就简单梳理一下用hexo搭建blog网站的流程吧。

Step1：下载好node（这是一个下载链接）

  • 下载LTS长期支持版的，然后直接按步骤一步步安装就好了

Step2：打开cmd，输入node -v和npm -v，查看一下刚刚安装的node和npm的版本

Step3：输入npm install -g cnpm --registry=https://registry.npm.taobao.org通过淘宝镜像源安装cnpm（因为cnpm比npm快），然后输入cnpm看看安装好了没

Step4：通过cnpm开始正式安装hexo博客，输入cnpm install -g hexo-cli开始安装

然后用hexo -v验证一下

这样hexo的安装就到此结束了，完结撒花，然后开始正式搭建博客

Step5：首先在C:\Users\user目录下(这个其实随意)新建一个文件夹myblog(这个也随意)，然后在cmd输入cd myblog进入myblog这个文件


Step6：在当前目录输入hexo init开始生成博客，然后就解放双手在那等完成就好了(可能时间会有点长，慢慢等~~)终于装完啦，让我们看看myblog文件夹里生成了点啥东西

Step7：看到生成了这些东西后，我们就可以启动博客了，方法极其简单，输入hexo s即可

启动成功，在本地生成了一个端口localhost:4000，我们来在浏览器中输入这个端口试试看

nice! 搞定了！一个简简单单普普通通的博客就搭建好了，收工。至于后面怎么在里面写博客，以及怎么上传到远端那就改日再议。88
